# LeafLine
Garden Website

Fullstack MERN application for plant care & guides, plant info & schedule management.

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: MongoDB

## Setup

### Backend
```bash
cd backend
npm install
npm run dev


#.env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret


#frontend 
cd frontend
npm install
npm run dev
